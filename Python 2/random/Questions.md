🔵Part 1: Simple Fundamentals (Basics)
Variables & Input/Output  
Write a program that takes a user’s name and age as input, then prints:
"Hello <name>, you are <age> years old."

Conditional Statements  
Check if a number entered by the user is even or odd.

Loops  
Print the multiplication table of a number (from 1 to 10).

Lists & Iteration  
Take 5 numbers as input, store them in a list, and print the largest number.

Functions  
Write a function factorial(n) that returns the factorial of a number using recursion.



🔵 Part 2: Logic‑Testing Challenges (Advanced)
String Manipulation  
Write a program to check if a string is a palindrome (same forwards and backwards).

Dictionary & Frequency Count  
Count the frequency of each word in a given sentence and print the results.

File Handling  
Write a program that reads a text file and prints the number of lines, words, and characters.

Object‑Oriented Programming  
Create a class BankAccount with methods to deposit, withdraw, and check_balance.

Algorithmic Thinking
